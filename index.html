Privacy Policy

Fractal Tree is a static, client-side visual generator. Core rendering and controls run in your browser.

No personal data collected

This app does not include account signup, user profiles, contact forms, or analytics trackers.

No cookies by this app

The application does not set or read cookies for app features.

No persistent app storage

Tree parameters are kept in runtime memory only. This app does not write settings to localStorage or sessionStorage.

Clipboard access is user-triggered

Clipboard writes happen only when you press the copy seed button.

Hosting

This site is deployed on Vercel. Vercel may process technical request metadata (such as IP address and user-agent) to deliver and secure the service. See Vercel Privacy Policy.

Third-party resources

The interface uses local system fonts and a self-hosted p5.js build served from this site. No external runtime script or font CDN is required by the app.

Data retention

Generated PNG files are downloaded directly to your device when you click save. No tree data is sent to an application backend.
